# Assets

Drop `install.gif` here — a short screen recording of installing the integration
via HACS and adding it from **Settings → Devices & Services**. The repository
README references it as `assets/install.gif`.

Suggested capture: ~10–20s, HACS custom-repo add → install → restart → add
integration → enter host. Tools: macOS `Cmd-Shift-5` (record) then convert to GIF
(e.g. `ffmpeg`/`gifski`), or Kap / LICEcap / Peek.
